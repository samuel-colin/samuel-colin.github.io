# samuel-colin.github.io

Personal web site :)

## Tech

[VueJS 3.2.13](https://vuejs.org/) and [Materialize 3.0.1](https://materializecss.com/)

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
